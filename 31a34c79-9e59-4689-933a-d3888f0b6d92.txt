NAC 장비는 네트워크의 엣지에 배치되어 네트워크 접근을 제어한다. 일반적인 네트워크 배치 구조는 [사용자 기기] -> [액세스 포인트/엣지 스위치] -> [NAC 장비] -> [코어 스위치] -> [방화벽] -> [라우터] -> [인터넷] 이 된다. 액세스 포인트 (Access Points)는 무선 네트워크를 이용하는 경우, 와이파이 액세스 포인트가 NAC 장비 앞에 존재한. 엣지 스위치 (Edge Switches)의 경우는 유선 네트워크를 사용하는 경우, 사용자 기기들이 연결된 스위치가 NAC 장비 앞에 위치한다. 일반적으로 NAC 장비는 네트워크의 엣지나 코어에 배치되어 네트워크 접근을 제어하고, 다른 네트워크 장비와 협력하여 전테 네트워크의 보안을 유지한다.

엣지 스위치는 케이블을 통해 기기들을 연결한다. 컴퓨터나 프린터 같은 기기들이 네트워크에 연결할때 해당 장비에 연결하게 된다. 여러개의 이더넷 포트를 가지고 있다. 각 기기는 케이블을 통해 엣지 스위치에 연결되고, 엣지 스위치는 이러한 기기들을 네트워크에 접속할 수 있도록 지원한다. 엣지 스위치는 각 기기들로부터 데이터를 받아서 네트워크의 다른 부분으로 전달한다.

무선 네트워크는 케이블이 없이 무선 신호를 통해서 기기들을 연결하는 방식이도, 노트북 태블릿 스마트폰 같은 기기들이 무선으로 인터넷 접속시 사용한다. 액세스 포인트가 기기들롤부터 와이파이 신호 및 데이터를 받아서 유선 네트워크로 전달한다. 액세스 포인트는 보통 여러 곳에 배치되어 넓은 영역을 커버할 수 있다.

NAC 장비는 유무선 네트워크 모두 네트워크에 접속하려는 기기들을 검사하고, 네트워크에 안전하게 접속할 수 있도록 지원한다. NAC는 다양한 기준으로 네트워크에 접속하려는 기기가 안전한 상태인지를 판단한다. 개략적으로는 사용자 인증, 기기 상태 검사, 보안 설정 검사, 네트워크 위치, 접속 시간, 기기 유형들을 판별한다. 사용자 인증시에는 ID 및 비밀번호는 문자 인증 코드등을 받는다. 기기 상태 검사의 경우에는 장비에 보안 소프트웨어가 설치되어 있는지를 확인한다. 보안 설정 검사에는 기기내에 방화벽이 활성화되어 있는지를 확인한다. 네트워크 위치의 경우에는 특정 위치에서 접속을 시도하려는 경우 접속을 차단한다. 기기 유형의 경우에는 특정 유형의 기기로 접속하려는 경우에는 차단한다. NAC의 판단하에 기기가 안전하지 않다고 판단되면 기기의 네트워크 접속을 막는다.

일반 가정용으로 쓰이는 공유기는 엣지 스위치, 엣지 포인트, 라우터, 방화벽 기능을 모두 갖춘 통합 장비라고 볼 수 있다. 따라서 가정 네트워크에서는 공유기 하나만으로 여러 기기를 유무선으로 연결하고, 인터넷에 접속이 가능하다. 

ISP (Internet Service Provider)는 인터넷을 제공회주는 회사를 의미한다. ISP는 다양한 방법으로 인터넷을 제공하는데 전화선을 이용한 DSL, 케이블을 사용한 케이블 인터넷 광섬유를 사용한 광대역 인터넷 등이 존재한다. ISP에 요금을 네고 인터넷 서비스를 가입하고 나면 ISP는 대상자의 집에 인터넷 신호를 쏴주게 된다 이 인터넷 신호를 받는 주체 장비가 모뎀이되고, 모뎀에 연결된 공유기를 통해서 여러 기기가 인터넷을 사용할 수 있게 된다. 모뎀은 다른 네트워크로 부터 인터넷 신호를 처음 주고받는 매개체의 역할을 한다. ISP로부터 받은 신호를 우리가 사용할 수 있도록 변환해주는 역할을 한다. 왜냐면 전화선, 케이블, 광섬유에 따라서 전송 신호 형태가 다르기 때문이다. 

모뎀은 케이블을통해 전송되는 아날로그 신호를 디지털 신호로 변환하거나 디지털 신호를 아날로그 신호로 변환해주는 장치다. ISP로부터 받는 인터넷 신호를 우리가 사용할 수 있는 형태로 변환해주는 역할을 하고 인터넷 신호가 네트워크 안으로 들어올 수 있게 해주는 인터넷 서비스의 첫 단계라고 볼 수 있다.

라우터는 네트워크간 데이터 패킷을 전달하고, 가장 데이터를 효율적인 경로로 전달해주는 역할을 한다. 로컬 네트워크간 기기들의 통신 정보를 관리하고 기기들이 외부 인터넷에 접속할 수 있도록 지원한다. 게이트웨이랑 개념적으로 헷갈릴 수 있는데 게이트웨이는 네트워크에서 다른 네트워크로 이동할 때 반드시 거쳐야하는 엔드포인트 주소의 개념으로 IP 주소를 의미한다. 하지만 라우터는 다른 네트워크간에 통신을 할 수 있도록 지원해주는 물리적인 장치를 의미한다
